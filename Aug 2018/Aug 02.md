# Python and Django

## 1-9. Lists

li : Lists.
ol : 순서 있는 Lists (=/= ul)
ul : 순서 없는 Lists

```
<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
```

Lists 안에 Lists 추가 가능!

## 1-10. Divs and Spans

div : division, 레이아웃을 설계하는 용도
span : div와 비슷함. 둘다 CSS와 함께 쓰인다.
